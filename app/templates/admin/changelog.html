What's New

Development changelog for CoachPrash.com

15 May 2026 Phase 3

Content, SEO, Security & Launch Polish

Demo content for 5 subjects, security hardening, SEO infrastructure, UI polish, and performance improvements.

Security

  • Flask-Limiter rate limiting on login (5/min), register (3/min), contact (2/min), and API endpoints (60/min)
  • HTML sanitization via nh3 on all admin content inputs
  • Security headers: CSP, X-Frame-Options, X-Content-Type-Options, X-XSS-Protection, Referrer-Policy
  • Custom error pages for 404, 403, and 500 errors
  • Context processor wrapped in try/except for 500 page resilience

Demo Content (5 Subjects)

  • Mathematics — Algebra 1: Linear Equations (4 concepts, 27 problems)
  • Physics — Honors Physics: Newton's Laws (4 concepts, 25 problems)
  • Chemistry — Honors Chemistry: The Mole Concept (3 concepts, 20 problems)
  • Computer Science — AP CS A: Arrays & ArrayLists (4 concepts, 24 problems)
  • Test Prep — SAT Math: Heart of Algebra (3 concepts, 20 problems)
  • All content follows qhsJSON format with hints, solutions, and callout boxes
  • Content auto-loaded during seed via content_loader utility

SEO

  • Dynamic robots.txt with sitemap reference
  • Auto-generated sitemap.xml covering all subjects, topics, free concepts, and blog posts
  • Meta description, Open Graph tags, and canonical URLs on every page
  • SVG favicon (CP monogram)

UI Polish

  • Callout box styling: key (green), warning (red), tip (blue)
  • Loading spinners on quiz submit, hint, and solution buttons
  • Empty state messages for progress dashboard and blog
  • Card hover lift animations

Performance

  • Blog pagination (20 per page) with reusable pagination partial
  • Static asset cache-busting via ?v= query parameter
  • Cache-Control headers for static files (1 year max-age)
  • Database indexes on slug, FK, and is_active columns

Seed Data Updates

  • 8 testimonials (up from 5) with varied subjects and grades
  • 4 blog posts (up from 2): AP Calc, SAT prep, physics problem-solving, why CS matters
  • 2 demo students: demo.student@example.com (premium) and demo.free@example.com (free)

15 May 2026 Phase 2A

Hardening & Content Pipeline

Resource model for teaching materials, image hosting via Railway Storage Bucket, qhsJSON standard, FRQ support, and content generation pipeline.

Teaching Resources

  • New Resource model for Google Slides, Docs, video links, PDFs, and external links
  • Resources attach at Topic or Subject level with freemium gating
  • Google Slides/Docs auto-converted to embed URLs and rendered as iframes
  • Admin CRUD for managing resources
  • Embedded slides section on topic overview page

Image Hosting

  • Railway Storage Bucket integration (S3-compatible, included in Hobby Plan)
  • Admin image upload page with bucket key display for qhsJSON
  • Template filter converts data-bucket-key attributes to presigned URLs at render time
  • Presigned URLs (1hr expiry) provide freemium gating — premium images only accessible to authorized users

qhsJSON Standard

  • Defined CoachPrash qhsJSON format — custom JSON for questions, hints, and solutions
  • Supports MCQ, fill-in-blank, and free response (FRQ) problem types
  • Images supported in questions, hints, AND solutions via data-bucket-key
  • Hints support cost_points for point deduction on reveal
  • JSON validation button on bulk import page (AJAX pre-check)
  • Updated format documentation on import page

Free Response Questions (FRQ)

  • New frq problem type
  • Students write answers in a textarea
  • Model answer displayed after submission
  • Honor-system self-grading: student marks their own answer as correct/incorrect
  • Self-grade counts toward quiz score (3/3 possible if FRQ marked correct)
  • Attempts logged for coach review

Admin Enhancements

  • Resources management page linked from admin dashboard
  • Image upload page linked from admin dashboard
  • Prompt template for Claude content generation (COACHPRASH_BUILD_PROMPTS.md)

Concept Explainers

  • Topic overview page showing all concepts with progress indicators and access tier badges
  • Full concept detail pages with KaTeX math rendering
  • Reading progress bar that fills as you scroll through a concept
  • Previous / Next concept navigation
  • Estimated reading time badges

Practice Quiz Engine

  • Interactive quizzes with multiple choice (MCQ) and fill-in-the-blank problems
  • AJAX-based answer checking — no page reloads, answers validated server-side only
  • Confetti animation on correct answers
  • Progressive hint system with slide-down reveal
  • Step-by-step solution accordion (premium only)
  • Built-in timer tracking time spent per problem
  • Results screen with score, accuracy, and time summary
  • Retry option to redo the same problem set

Freemium Content Gating

  • Server-side access control — not just UI hiding
  • Free tier: free-tier concepts, first 3 problems per set, first hint only
  • Premium tier: all concepts, all problems, all hints, step-by-step solutions, progress tracking
  • Premium teaser overlay with blur effect on gated content
  • Gentle sign-up prompts for unauthenticated users

Student Progress Dashboard

  • Stats cards: concepts completed, in progress, problems attempted, accuracy, study streak
  • Per-subject progress bars showing completion percentage
  • Recent activity timeline with correct/incorrect indicators
  • Automatic progress tracking when viewing concepts and completing quizzes

Admin Enhancements

  • Practice analytics on dashboard: daily/weekly attempts, overall accuracy
  • Most attempted and hardest problems tables
  • This changelog page

Bug Fixes

  • Upgraded Flask-Login from 0.6.2 to 0.6.3 to fix Werkzeug 3.1 compatibility (url_decode import error)
11 May 2026 Phase 1

Foundation

Complete project scaffolding, database schema, authentication, admin panel, all public pages, and Railway deployment.

Project Architecture

  • Flask app factory with Blueprint-based architecture
  • PostgreSQL database with 15 tables (UUID primary keys, timestamps)
  • SQLAlchemy ORM with Flask-Migrate for schema management
  • Single clean initial migration covering the complete schema
  • Custom CSS design system — Bold & Academic theme (navy, crimson, gold)
  • Google Fonts: Playfair Display (headings), Inter (body), JetBrains Mono (code)
  • KaTeX CDN integration for LaTeX math rendering
  • Vanilla JavaScript — no frameworks

Authentication

  • Email/password registration and login with Flask-Login
  • Google OAuth sign-in via Authlib (OpenID Connect)
  • Automatic account merging when Google email matches existing account
  • Access code system for tier upgrades (free/premium)
  • Case-insensitive email handling throughout

Public Pages

  • Home page with hero section, subject cards, testimonials, and CTAs
  • About page with coach bio and teaching philosophy
  • Subjects catalog with topic browsing
  • Contact form with database storage
  • Testimonials page with star ratings
  • Blog/Resources section with published/draft support
  • Responsive navigation with mobile toggle
  • Flash message system

Admin Panel

  • Dashboard with student count, premium count, concept count, problem count
  • Student management with search, tier editing, activation/deactivation
  • Full content CRUD: subjects, topics, concepts
  • Access code management with generation, activation, deactivation
  • Blog post management with publish/draft workflow
  • Testimonial management with featured flag
  • Contact message inbox with read/unread status

Deployment

  • Railway deployment with Gunicorn (2 workers)
  • Health check endpoint at /health
  • ProxyFix middleware for correct HTTPS URL generation behind load balancer
  • Custom domain: coachprash.com via Cloudflare (CNAME flattening)
  • www → root 301 redirect
  • Database seed script with subjects, topics, testimonials, blog posts, access codes
  • DB reset endpoint for development (/run-drop-all)

Seed Data

  • 5 subjects: Mathematics, Physics, Chemistry, Computer Science, Test Prep
  • 26 topics across all subjects
  • 5 student testimonials
  • 2 blog posts (AP Calculus tips, SAT prep strategies)
  • 3 access codes: FREEACCS (free unlimited), PREM2026 (premium 10 uses), EXPIRED1 (expired)
